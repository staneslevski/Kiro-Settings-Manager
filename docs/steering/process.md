# Development Process for AI Workers

LLMs have a bias towards writing new code and new functionality. You should question whether this is the best approach. This document contains instructions on how you should execute tasks and which course of action is preferred if several are available. 

## Refactoring existing code is preferred

1. Check whether the task that you are working on can be achieved by refactoring existing code instead of writing new code. If you can do it by refactroing, you should refactor.
2. Instead of creating a new file with a script to perfom an operation on the repository, you should check whether you can do this work manually using existing tools.
